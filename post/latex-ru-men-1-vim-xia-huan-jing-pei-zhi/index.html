latex入门(1)——vim下环境配置

2019-05-28

入坑latex，告别自己垃圾的手写体笔记

目标：使用Latex编辑Latex文档，并可以实时预览，主要是被震撼(刺激)到了.

配置步骤

  1. vimtex安装
  2. texlive安装 bash sudo apt-get install texlive-full 这里不一定要安装full版本，但是中文排版需要专门的宏包，full版本可以省去很多麻烦
  3. vim-latex-live-preview插件安装
  4. zathura安装 zathura是一个pdf预览软件，支持latex编辑的自动刷新，可以使用vim-latex-live-preview支持的其他软件，在vimrc文件中指名即可

.vimrc文件中最终相关配置

" Plugins will be downloaded under the specified directory.
call plug#begin('~/.vim/plugged')

" Declare the list of plugins.
Plug 'lervag/vimtex'
let g:tex_flavor = 'latex'
let g:vimtex_version_check = 0
let g:vimtex_view_method='zathura'

Plug 'xuhdev/vim-latex-live-preview', { 'for': 'tex' }
let g:livepreview_previewer = 'zathura'
" 注意这里指名编译使用xelatex，否则默认使用pdflatex会报错
let g:livepreview_engine = 'xelatex'
"let g:livepreview_cursorhold_recompile = 0

" List ends here. Plugins become visible to Vim after this call.
call plug#end()


filetype plugin indent on
syntax on

测试安装

新建一个空目录，使用vim编辑一个简单的tex文件(保证语法正确)，使用:LLPStartPreview命令进行编译，正常情况下可以弹出pdf预览界面，表示安装成功。

注意:一定是一个空目录，否则会报Failed to compile bibliography 或 Failed to compile的错。参见：https://github.com/xuhdev/vim-latex-live-preview/issues/21#issuecomment-322240236
